Data Flow — Filter to Render

1
User changes a filter in FilterBar Period · Warehouse · Category · Order Status
2
useFilters composable updates singleton refs Reactive state shared across all views; getCurrentFilters() returns the active set
3
View calls api.getInventory(filters) src/api.js builds URLSearchParams and issues axios.get to /api/inventory?warehouse=…&category=…
4
FastAPI endpoint applies filters server-side apply_filters() and filter_by_month() iterate the in-memory list — no client-side filtering of the response
5
Pydantic validates each item List[InventoryItem] etc. — auto-serialised to JSON with type guarantees
6
Vue stores raw data in refs e.g. allOrders, inventoryItems — never mutated after fetch
7
Computed properties derive display data Totals, grouped sums, currency & i18n formatting — re-evaluated reactively
8
Template renders tables, SVG charts, modals v-for keyed by sku/month/id; details surfaced in modal components
